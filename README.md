# 🚌 NgeBus
### Sistem Pemesanan Tiket Bus Online

**NgeBus** adalah aplikasi pemesanan tiket bus berbasis web yang memungkinkan pengguna mencari jadwal perjalanan bus, memilih kursi, dan melakukan pemesanan tiket secara online dengan mudah.

Proyek ini dibangun menggunakan arsitektur **Fullstack Modern**:

- ⚡ **Frontend** : Vue 3 + Vite
- 🚀 **Backend** : Golang
- 🗄 **Database** : MySQL / PostgreSQL
- 🔐 **Authentication** : JWT

---

# ✨ Fitur Utama

### 👤 Pengguna
- Registrasi akun
- Login pengguna
- Melihat daftar bus
- Melihat jadwal perjalanan
- Memilih kursi
- Memesan tiket bus
- Riwayat pemesanan tiket

### 🛠 Admin
- Manajemen bus
- Manajemen jadwal bus
- Melihat data pemesanan
- Mengelola pengguna
